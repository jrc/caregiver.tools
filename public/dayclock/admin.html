Day Clock Admin

Bookmark this page. This is the only way that you, the caregiver, can customize the Day Clock later. There are no user accounts or passwords.

Do not share this page’s web address. It contains a unique code for your Day Clock.

Customize Your Clock

You can set a custom message to be displayed at the bottom of the Day Clock.

You can write {{Date:YYYY-MM-DD}} with a future date and it will be shown in natural language, such as “tomorrow” or “in 5 days”.
